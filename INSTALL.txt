Before installation make sure that all requirements are met:

- PHP 5.4+
- Mysql 5+
- Apache 1+

Extensions:
- Curl
- OpenSSL
- Mb string
- Apache Mod Rewrite
- ZipArchive

MANUAL INSTALLATION
-------------------

1. Download and extract to your hosting directory.
2. Create folder "vendor/gplcart/libraries" in your root directory (i.e where index.php is)
3. Download 3th party libraries from https://github.com/gplcart/libraries, extract and copy all folders to "vendor/gplcart/libraries"
4. Go to your http://site.com or http://localhost/site and follow the instructions
